# 🌐 Personal Portfolio Website - Nikhil Kumar Sutail

Welcome to my **Personal Portfolio Website** – a clean, modern, and responsive portfolio built using **HTML**, **CSS**, and **JavaScript**.

This website is designed to reflect who I am as a professional — showcasing my experience, projects, and passion for web development, all wrapped up in an elegant user interface.

## 👤 About Me

I’m **Nikhil Kumar Sutail**, a dedicated **Web Developer**, **Graphic Designer**, and **Assistant Teacher** based in India. I enjoy turning complex problems into simple, beautiful, and intuitive designs. When I'm not coding, I’m mentoring students and guiding them into the world of programming.

## 🚀 Features

- ✅ Fully responsive layout for all screen sizes
- 🎨 Stylish, clean UI/UX using modern CSS techniques
- ⌨️ Typing animation with [Typed.js](https://github.com/mattboldt/typed.js/)
- 🧠 About Me section with a profile overview
- 💼 Project section with sample projects (e.g., Amazon Clone, Tech Savvy)
- 📫 Contact section with email & LinkedIn
- 📱 Social media icons (Font Awesome integration)
- 🔗 Easy navigation across sections
- 🧾 Footer with copyrights

## 🛠 Tech Stack

- **HTML5** – Page structure
- **CSS3** – Styling, responsiveness, layout
- **JavaScript** – Dynamic typing animation
- **Font Awesome** – Icons and social links
- **Typed.js** – Typewriter effect
